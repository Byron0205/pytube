Aspectos a tener en cuenta:
1. Tener python instalado en tu ordenador(te recomiendo instalar python 3.7 o 3.8 para evitar problemas de compatibilidad)
2. tener pip actualizado a la ultima versión antes de instalar pytube

Para instalar pytube en el cmd de windows pegar lo siguiente:
python -m pip install git+https://github.com/nficano/pytube
